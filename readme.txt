Plugin to help you sell your products online using the Koi eCommerce platform.

== Description ==

This plugin is designed for people who have an existing site and want to add an online store to it.
It is not a complete eCommerce solution, but rather a plugin that will help you sell your products online using the [Koi eCommerce](https://koi.app) platform.

Products are added to your site using the [Koi](https://koi.app), and then displayed on your site using this plugin.

See the [Koi eCommerce](https://koi.app) website for more information.

== Frequently Asked Questions ==

= How do I list my products =

You can register for an account at https://koi.app and then list your products using the Koi eCommerce platform.
